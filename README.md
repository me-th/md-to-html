# md-to-html
simple markdown to html converter written purely in svelte

## what works
-images
-headers
-paragraphs
-injecting html code directly
-live preview of html generated from markdown
-copying html code
-hyperliks
-images inside hyperlinks

### what doesn't work
almost everything else
